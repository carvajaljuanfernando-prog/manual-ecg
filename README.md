# Manual Interactivo de Electrocardiografía

Recurso educativo gratuito para aprender a leer e interpretar electrocardiogramas (ECG), dirigido a estudiantes de ciencias de la salud. Va de lo más simple a lo más complejo, con trazos reales dibujados sobre papel milimetrado, práctica guiada y exámenes de comprensión.

## 🔗 Ver el manual

Una vez publicado con GitHub Pages, estará disponible en:

```
https://TU-USUARIO.github.io/TU-REPOSITORIO/
```

(Reemplaza `TU-USUARIO` y `TU-REPOSITORIO` por los tuyos.)

## ✨ Qué incluye

- **16 módulos progresivos:** desde "qué es un ECG" hasta infartos, bloqueos, arritmias ventriculares y trastornos electrolíticos.
- **+25 trazos interactivos** generados sobre papel milimetrado clásico (no son imágenes: se dibujan con un motor propio).
- **Calculadora de frecuencia** y **selector de ritmos** en vivo.
- **Módulo de práctica** con retroalimentación inmediata.
- **Examen de comprensión** con banco de 55 preguntas y 5 modalidades (general, ritmos, bloqueos, isquemia, fundamentos); cada intento toma preguntas distintas y baraja las opciones.
- **Funciona sin conexión** una vez cargado y se adapta a **móvil, tablet y computadora**.

## 🚀 Cómo publicarlo (GitHub Pages)

1. Sube `index.html` (y este `README.md`) a tu repositorio.
2. En el repositorio, ve a **Settings → Pages**.
3. En **Build and deployment → Source**, elige **Deploy from a branch**.
4. Selecciona la rama **main** y la carpeta **/ (root)**. Guarda.
5. Espera ~1 minuto y recarga: aparecerá la URL pública del sitio.

## 📦 Tecnología

Un solo archivo HTML autónomo: HTML + CSS + JavaScript (vanilla), sin dependencias externas ni servidor. Cualquier navegador moderno lo ejecuta.

## ⚠️ Aviso

Material **educativo**, no diagnóstico. No sustituye la formación clínica supervisada ni el juicio profesional. Un ECG se interpreta siempre en el contexto clínico del paciente.

## 📚 Referencias

Recurso independiente de elaboración propia, inspirado en la enseñanza clásica de la electrocardiografía. Para profundizar se recomienda consultar textos de referencia y guías clínicas vigentes.
